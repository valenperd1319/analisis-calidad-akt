# Análisis de Calidad de Pintura — AKT Motos

Herramienta de análisis y seguimiento de defectos de calidad para el Área de Desarrollo de Producto.

## Cómo usar

1. Sube el master de averías (.xlsx) en el panel izquierdo
2. Dale un nombre al período (Ej: "Q1 2026")
3. Analiza por proveedor, defecto o pieza
4. Compara períodos para detectar reincidencias
5. Registra y da seguimiento a las acciones tomadas

## Funcionalidades

- 📊 Resumen general con KPIs y distribución STD-001
- 🏭 Análisis por proveedor con recomendaciones según tipo de proceso
- 🔍 Trazabilidad por defecto y modelo
- 🔧 Ranking de piezas críticas
- 📈 Comparación entre períodos con alertas de reincidencia
- ✅ Registro y seguimiento de acciones correctivas

## Deploy

Conecta este repositorio a [Streamlit Cloud](https://streamlit.io/cloud) para tener la app disponible en una URL pública.
